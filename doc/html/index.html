Infomap HTML Documentation

Note that in addition to this HTML documentation, man pages for the various Infomap programs are provided.

  • Infomap User Manual — Explains how to compile and install the Infomap software and how to use its standard functionality.
  • Infomap Implementation Guide — A more detailed guide to the underlying implementation of the Infomap software; intended for those who wish to modify the code or use it in nonstandard ways.
  • Algorithm description — Explains the algorithms and theory behind the software.
  • Infomap Tutorial — COMING SOON Illustrates the use of the Infomap NLP software via worked examples on an actual corpus (provided with the software).
